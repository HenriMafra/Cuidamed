# Changelog

## [1.0.0] - 2026-03-11

### Adicionado
- Interface CLI com menu interativo
- Funcionalidade de adicionar medicamentos com nome, horário e doses por dia
- Funcionalidade de listar todos os medicamentos cadastrados

- Funcionalidade de buscar medicamento por nome ou substring
- Persistência de dados em arquivo JSON local
- Testes automatizados cobrindo todos os fluxos principais
- Linting com ruff
- Pipeline de CI com GitHub Actions
